# Is This Copyright?
## What is this about?
Is This Copyright? is the website that you can check the music that is copyright, or public domain (which mean you can use the music).

## How is this work?
Check over ~500 songs that is copright or not, but if you not find the music, you can upload one of these audio files: (*MP3 · WAV · OGG · FLAC · AAC · M4A · OPUS · WMA · AIFF*).
